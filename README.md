# BILINK CCCES Official Website

React + Vite + TypeScript + Tailwind CSS implementation for `bilinkccces.ca`.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Stack

- React 18
- Vite 5
- TypeScript
- Tailwind CSS
- React Router

## Deployment

Build output is generated in `dist/` and can be deployed with GitHub Pages workflows.
